# Infographics & Visual Diagrams

This folder contains all visual diagrams, architecture illustrations, and infographics for the prolog-reasoning-v2 project.

## Naming Convention

All infographics follow a **numbered prefix** format to show iteration and ordering:
- `01-architecture-overview.png` - First architecture diagram
- `02-pipeline-flow.png` - Second diagram (example)
- `03-validation-layers.png` - Third diagram (example)

This makes it easy to:
1. Track multiple versions of related diagrams
2. Order them by topic or progression
3. Reference them in documentation with predictable names

## Creating New Infographics

### Recommendations:
1. **Use Mermaid diagrams** (embedded in markdown when possible)
2. **Use AI tools** (Gemini, DALL-E, etc.) for complex visual designs
3. **Export as PNG** for consistency
4. **Follow naming convention**: `NN-descriptive-name.png`

### Format Suggestions:
- Architecture & flow diagrams → Mermaid or Lucidchart
- Performance comparisons → AI-generated for visual appeal
- Data pipeline visualization → Mermaid flowchart
- System interaction diagrams → Mermaid sequence diagram

### Include in README:
Once created, reference new infographics in:
- `README.md` section headers
- `ARCHITECTURE.md` for technical details
- `training/` course materials for educational content
