## 📖 Available Courses

### **Level 1: Foundations**

#### [01 - LLM Memory Magic: Never Forget Again!](01-llm-memory-magic.md)
**Duration:** 30 minutes | **Domain:** LLM Fundamentals | **Level:** Beginner

Learn why LLMs forget everything and how neuro-symbolic AI gives them perfect memory. Perfect first course if you're new to AI.

**Topics:**
- LLM memory limitations
- How neuro-symbolic AI works
- Practical examples (family trees, permissions, game rules)
- Getting started with code

**Best For:** Anyone just discovering AI, students, curious developers

#### [02 - Knowledge Bases 101: Facts That Don't Lie](02-knowledge-bases-101.md)
**Duration:** 25 minutes | **Domain:** Knowledge Representation | **Level:** Beginner

Learn how to structure information so AI gets it right every time. Discover the difference between guessing and knowing, and why this matters for safety-critical applications.

**Topics:**
- What makes a reliable knowledge base
- Facts vs. rules (and when to use each)
- Preventing "false certainty" errors
- Prolog syntax and structure
- Building your first knowledge base

**Best For:** Students of Course 1, anyone building AI systems, developers curious about logic programming

#### [03 - Learning from Failures: When AI Gets It Wrong](03-learning-from-failures.md)
**Duration:** 20 minutes | **Domain:** Error Handling & Debugging | **Level:** Beginner

Learn to read error messages like a detective, understand why systems fail, and turn mistakes into learning opportunities. Essential for anyone deploying real AI systems.

**Topics:**
- Why errors are actually good (they prevent worse problems)
- Decoding error messages (what they really mean)
- "Did you mean?" suggestions and typo recovery
- The error recovery loop (try → fail → understand → fix → retry)
- Preventing errors before they happen
- Error patterns across different systems

**Best For:** Anyone who's gotten a confusing error message, developers building AI products, people learning system design
