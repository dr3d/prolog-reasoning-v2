# Idea Inbox

Use this file to capture fast thoughts before they evaporate.

## Capture Rule

Add first, sort later. No polish required.

## Quick Entry Format

`- [ ] YYYY-MM-DD HH:MM ET | area:<tag> | idea: <one-liner> | why: <value/risk in one line>`

Example:

`- [ ] 2026-04-09 08:10 ET | area:prethink | idea: add confidence floor before fact ingestion | why: lower false writes on uncertain turns`

## Triage Buckets

Copy items from Inbox into one of these during review.

- `Now`: directly supports current branch objective.
- `Next`: valuable but not blocking current branch.
- `Parked`: keep for later; do not let it distract active work.
